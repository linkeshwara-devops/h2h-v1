## Direct Farm-to-Hotel Supply Chain Marketplace

Harvest2Hotel is India's first digital marketplace connecting farmers directly with hotels, restaurants, and bulk buyers. Eliminate middlemen, get fresher produce, and ensure fair prices for everyone.

## 🚀 Features

### For Farmers
- ✅ List and manage products with real-time inventory
- ✅ Receive and manage bulk orders
- ✅ Secure payment settlements
- ✅ Sales analytics and insights

### For Hotels/Restaurants
- ✅ Browse fresh produce by categories
- ✅ Place bulk orders
- ✅ Create recurring subscriptions (daily/weekly/monthly)
- ✅ Track deliveries in real-time
- ✅ Purchase analytics

## 🛠️ Technology Stack

- **Backend**: Flask (Python 3.11)
- **Database**: SQLite (Development) / PostgreSQL (Production)
- **Frontend**: HTML5, CSS3, JavaScript
- **Authentication**: Session-based with Werkzeug
- **Deployment**: Docker, Gunicorn

## 📋 Prerequisites
